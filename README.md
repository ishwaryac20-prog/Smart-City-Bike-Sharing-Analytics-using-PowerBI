## 📌 Project Overview

This project analyzes a **Smart City Bike Sharing** dataset using **Microsoft Excel** and **Power BI**. The dataset was cleaned and transformed in Excel, followed by data modelling, DAX calculations, and interactive dashboard development in Power BI.

The dashboard provides valuable insights into bike station availability, station status, bike stand capacity, banking facilities, bonus-enabled stations, and geographical distribution.

---

## 🎯 Project Objectives

- 🧹 Clean and transform raw data using Microsoft Excel.
- 📊 Build an interactive Power BI dashboard.
- 📈 Perform DAX calculations for KPI analysis.
- 🚲 Analyze bike station performance and availability.
- 📍 Visualize station locations using maps.
- 💡 Generate meaningful business insights.

## 💡 Key Insights

- 📍 Total Bike Stations: **2,856**
- 🚴 Available Bikes: **19,646**
- 🅿 Total Bike Stand Capacity: **57,203**
- 📊 Bike Availability: **34.34%**

### 🔍 Observations

- Most bike stations are operational.
- Bike availability varies across different cities.
- Banking and Bonus facilities are available only at selected stations.

### 🚀 Recommendations

- Increase bike allocation in high-demand cities.
- Monitor stations with low bike availability.
- Expand Banking-enabled stations.
- Improve Bonus-enabled station coverage.
